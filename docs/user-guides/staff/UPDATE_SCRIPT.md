# Script cập nhật các file nhân viên

Các file đã được copy từ quản lý/ sang nhân viên/ nhưng cần cập nhật thủ công:

1. Sửa tiêu đề: `# QUẢN LÝ XXX - MANAGER` → `# QUẢN LÝ XXX - STAFF`
2. Sửa phần "Quyền truy cập" để thêm Môi giới với capabilities
3. Thay `/staff/` thành `/staff/`
4. Thay "menu Nhân viên" thành "menu Nhân viên"
5. Cập nhật ngày: 2025-11-03
